# Georgian Cartographers Association — Website

საქართველოს კარტოგრაფთა ასოციაციის (სკა) ოფიციალური ვებ-გვერდი.

სკა არის პროფესიული და სამეცნიერო ორგანიზაცია, რომელიც 1998 წელს დაარსდა საქართველოში კარტოგრაფიის, გეოდეზიისა და გეოინფორმაციული მეცნიერებების განვითარების მიზნით.

🌐 **[gc.qgis.ge](https://gc.qgis.ge/ka)**

## ტექნოლოგიები

- [Next.js 16](https://nextjs.org) (App Router)
- TypeScript
- Tailwind CSS v4
- [next-intl](https://next-intl.dev) — ორენოვანი ინტერფეისი (ქართული / English)
- Leaflet — ინტერაქტიული რუკებისთვის
- Vercel — დეპლოი

## გაშვება ლოკალურად

```bash
npm install
npm run dev
```

გახსენი [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm start
```
